# Desktop Expense Tracker

A local desktop expense tracker built with Electron, Node.js and SQLite.

The app stores all date locally on the user's computer and does not rely on any cloud services.
Built as a learning project to explore desktop app development with Electron

## Tech Stack
- Electron
- Node.js
- SQLite
- HTML / CSS / JavaScript

## Features
- Add and manage expense categories
- Track expenses locally
- Minimal but complete analytics view
- Persistent local storage using SQLite

## Run Locally

```bash
npm install
npm start
```

## Notes 
- All info is saved locally and retrieved with every restart
- No internet connection is required to use the app
